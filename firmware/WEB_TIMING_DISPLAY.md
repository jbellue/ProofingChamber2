## API Response Example

```json
{
  "temperature": 28.5,
  "mode": "heating",
  "isHeating": true,
  "isCooling": false,
  "proofingStartTime": 1738599600,
  "proofingElapsedSeconds": 932
}
```

Or during cooling:

```json
{
  "temperature": 5.2,
  "mode": "cooling",
  "isHeating": false,
  "isCooling": true,
  "coolingEndTime": 1738607700,
  "coolingRemainingSeconds": 8144
}
```

## Update Frequency

- Temperature: Updates every 2 seconds
- Mode: Updates every 2 seconds
- Timing information: Updates every 2 seconds (live countdown)
- All updates happen via AJAX polling to /api/status
